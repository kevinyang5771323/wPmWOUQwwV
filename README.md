# SpringBoot影院管理系统 java1197

## 项目概述

本项目是基于前后端分离的SpringBoot影院管理系统，提供管理员端和用户端两大部分，涵盖影院信息管理、电影管理、影厅管理、订单管理、用户管理等功能。

## 技术栈

- **推荐环境配置**：
  - IDEA
  - JDK 1.8
  - Maven
  - MySQL
- **后台**：
  - SpringBoot
  - MyBatis
  - MySQL
- **前台**：
  - Vue

## 功能模块

### 管理员端

- **影院管理**：
  - 影院信息管理
- **电影管理**：
  - 电影信息管理
  - 电影类管理
- **影厅管理**：
  - 影厅信息管理
  - 场次信息管理
- **订单管理**：
  - 订单信息管理
- **用户管理**：
  - 用户信息管理
- **权限管理**：
  - 角色信息管理
  - 资源信息管理

### 用户端

- 注册
- 登录
- 个人中心
- 影片查询
- 查看及选座
- 购票
- 支付
- 我的订单
- 影片榜单
- 正在热映
- 即将上映
- 热播影片

## 系统角色

- 管理员：负责管理后台各项数据和系统维护。
- 用户：实现个人信息的维护、电影查询、购票等操作。

## 运行环境

- 后端：基于SpringBoot的Java应用，需部署在支持JVM的服务器上。
- 前端：基于Vue的Web应用，需要现代浏览器支持。

## 目录结构

```
/docs              # 项目文档
/src               # 源码目录
  /main             # 主目录
    /java           # 后端Java代码
    /resources      # 资源文件，包括配置文件
  /test             # 测试代码
/front-end         # 前端代码，基于Vue
```

## 核心亮点

- **完善的权限管理**：实现了角色与资源的精细化管理。

## 部署步骤

1. 准备推荐环境配置。
2. 导入数据库文件，建立数据库环境。
3. 编译后端代码，并部署至服务器。
4. 编译前端代码，并部署至支持Vue的Web服务器。
5. 确保前后端可以正常通信。

_注：具体部署细节请结合项目文档和实际环境进行调整。_
